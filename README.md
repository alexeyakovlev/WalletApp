# WalletAPP

Пиложение для работы с кошельками. В приложении реализованы возможности управления состоянием(создание, удаление) и балансом(пополнение или снятие средств) кошелька

## Инструкция по запуску

1. Собрать JAR приложения
`mvn clean package -DskipTests`

2. Запустить всю систему
`docker-compose up --build`

3. Остановить систему
`docker-compose down`

4. Остановить и удалить volumes (данные БД)
`docker-compose down -v`
